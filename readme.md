
## TobyMaxham Database

SEE [Illuminate Database](https://github.com/illuminate/database)

Quickly use
```php
TobyMaxham\DBConnector::connect([
	'host'      => 'localhost',
	'database'  => 'database',
	'username'  => 'root',
	'password'  => 'password',
]);
```
and you'll be done.
